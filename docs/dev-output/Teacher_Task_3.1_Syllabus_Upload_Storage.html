Teacher Task 3.1 — Syllabus Upload API & Storage

Sprint: 3 — Syllabus & AI Ingestion | App: Teacher | Status: Done | Date: 2026-03-02 | SP: 2

Summary

Multipart file upload API for syllabus documents (PDF/Image/Text). Validates file type, MIME, and size (max 20 MB). Stores to local filesystem under per-class directory. Creates syllabus_document record with status pending for downstream AI processing.

API Endpoints

MethodPathPurpose

Validation Rules

RuleDetail
Allowed extensions.pdf, .jpg, .jpeg, .png, .webp, .txt

Storage path{SYLLABUS_STORAGE_PATH}/{classId}/{timestamp}_{sanitizedName}
